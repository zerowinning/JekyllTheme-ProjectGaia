Git LFS

![git的原理](/img/gitlfs.gif)

### 安装git lfs
git lfs install

### 添加大文件跟踪
git lfs track "*.psd"


### Make sure .gitattributes is tracked
git add .gitattributes

### 正常提交就可以了
git add file.psd
git commit -m "Add design file"
git push origin master
